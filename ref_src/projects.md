# 專案作品 (Projects)

## Web Application
### TradeGuard
- **專案類型**: Web Application / Tool
- **技術棧**: TypeScript, React 19, Vite, Tailwind CSS, Local Storage (No Backend)
- **開發週期**: 2025/11 - 2025/12
- **連結**: [GitHub](https://github.com/Lewsiafat/TradeGuard) | [Live Demo](https://lewsi.ddns.net/TradeGuard)
- **概述**: 專為加密貨幣交易者設計的輔助工具，透過強制執行交易前檢查清單來提升紀律，並提供完整的持倉追蹤與損益分析。
- **成效**:
  - 採用多 AI Agent 接力開發手法 (Gemini, Claude...)，大幅提升開發效率。
  - 實現無後端架構，資料完全本地化，確保隱私與響應速度。
  - 提供視覺化儀表板與詳細交易歷史，協助回測與優化策略。
  - 支援深色模式與響應式設計，優化長時間看盤體驗。

## AI / CLI 工具
### AICodeReviewCLI
- **專案類型**: AI 驅動的命令行代碼審查工具
- **技術棧**: Python, Design Patterns (Strategy/Factory), Multi-AI Integration
- **開發週期**: 2025/08 - 2025/10
- **連結**: [GitHub](https://github.com/Lewsiafat/AICodeReviewCLI)
- **概述**: 整合 Gemini、GPT、Claude、Grok 等多個 AI 模型的 CLI 工具，旨在解決人工 Code Review 效率瓶頸。
- **成效**:
  - 在 6 人團隊中實際導入應用，將審查時間從每月平均 4 天縮短至 2 天。
  - 導入客製化提示詞，提升 AI 審查的一致性。
  - 支援 Git 模式與檔案模式，靈活適應不同開發場景。
  - 純 Command Line 操作，無 GUI 介面，適應遠端開發環境。

## 開發工具
### WebAppTemplate
- **專案類型**: 多語言後端專案樣板生成器
- **技術棧**: Bash Script, Template Engine
- **開發週期**: 2025/09
- **連結**: [GitHub](https://github.com/Lewsiafat/WebAppTemplateeee)
- **概述**: 自動化專案初始化工具，支援多種後端框架與統一的前端架構。
- **成效**:
  - 支援 Python (FastAPI), Node.js (Express), Go (Gin) 三種後端選項。
  - 導入 SDD (Specification-Driven Development) 概念，自動生成 AI 開發指引文件。
  - 大幅降低新專案的環境建置成本。
